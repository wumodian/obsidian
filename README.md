## 仓库简介

这是一个个人学习与期末复习仓库，主要用于整理大学课程知识点、公式、习题解析、教师作业和考试重点。

目前仓库的核心内容是大学物理复习资料。笔记通过 Obsidian 双向链接连接章节、知识点、题目和复习总结，方便从整体复习路线逐步进入具体内容。

## 推荐复习方式

1. 从总复习提纲确定复习顺序。
2. 阅读章节解题框架和公式说明。
3. 通过知识点笔记理解公式和适用条件。
4. 独立完成章节习题，再查看答案和解析。
5. 按照概念、公式、方向、单位和题图整理错因。
6. 考前集中回顾公式总表、易错点和教师作业。

## 使用说明

本仓库可以直接作为 Obsidian 本地仓库打开，并使用 GitHub 进行多设备同步和版本备份。

笔记之间主要通过 `[[双向链接]]` 相互连接。打开一个知识点后，可以继续跳转到对应章节和相关题目。
