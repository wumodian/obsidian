# 光电效应

## 核心判断

光电效应是否发生由频率决定，光电子最大初动能由频率决定，光电流大小主要由光强决定。

## 必背公式与结论

- 爱因斯坦方程：$h\nu=W+E_{k,max}$。
- 截止频率：$\nu_0=W/h$。
- 红限波长：$\lambda_0=c/\nu_0=hc/W$。
- 最大初动能：$E_{k,max}=h(\nu-\nu_0)$。
- 遏止电压：$eU_s=E_{k,max}$。

## 适用条件

- $\nu<\nu_0$：无论光强多大、照射多久，都不产生光电效应。
- $\nu>\nu_0$：几乎立即逸出光电子。
- 增大光强使饱和光电流增大。
- 增大频率使最大初动能和遏止电压增大。

## 解题步骤

1. 找入射频率或波长，以及截止频率或逸出功。
2. 先判断是否满足阈值。
3. 用光电方程求最大初动能。
4. 用 $eU_s=E_{k,max}$ 求电压。
5. 问电流强弱时看光强和收集电压。

## 易错点

- 光强不能改变最大初动能。
- 频率不足时延长照射时间也无效。
- 达到遏止电压时电流为零，不是最大。
- 逸出功由金属材料决定。
