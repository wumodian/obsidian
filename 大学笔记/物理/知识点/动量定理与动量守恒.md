# 动量定理与动量守恒

## 核心判断

先明确系统和正方向，再用合外力冲量等于动量变化。只有系统合外力冲量为零，或某一方向合外力冲量为零，该方向动量才守恒。

## 必背公式与结论

- 动量：$\vec p=m\vec v$。
- 冲量：$\vec I=\int\vec Fdt$，恒力时 $\vec I=\vec F\Delta t$。
- 动量定理：$\vec I_{合外}=\Delta\vec p$。
- 动量守恒：$\sum m_i\vec v_i=\sum m_i\vec v_i'$。
- 完全非弹性碰撞：$m_1v_1+m_2v_2=(m_1+m_2)v'$。

## 适用条件

- 碰撞、爆炸、打击、反冲等短时过程常用动量。
- 某一方向外力可忽略时，只能说该方向动量守恒。
- 弹性碰撞同时满足动量守恒和机械能守恒；非弹性碰撞一般机械能损失。

## 解题步骤

1. 选系统，判断外力冲量。
2. 规定正方向，把速度写成带符号量。
3. 列初末动量方程。
4. 二维问题分 $x,y$ 方向列式。
5. 若求平均力，用 $\bar F\Delta t=\Delta p$。

## 易错点

- 速度反向时末动量要带负号。
- 动量变化是 $\vec p_2-\vec p_1$。
- 内力能改变各物体动量，但不改变系统总动量。
- 动量守恒是矢量守恒，不能只比较大小。
