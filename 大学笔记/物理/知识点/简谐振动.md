# 简谐振动

## 核心判断

简谐振动的本质是回复力或加速度与位移成正比且方向相反。判断时抓住 $a=-\omega^2x$ 或 $F=-kx$。

## 必背公式与结论

- $x=A\cos(\omega t+\varphi)$。
- $v=-A\omega\sin(\omega t+\varphi)$。
- $a=-\omega^2x$。
- $T=2\pi/\omega$，$f=1/T$。
- 弹簧振子：$\omega=\sqrt{k/m}$；小角度单摆：$\omega=\sqrt{g/l}$。
- 总能量：$E=\frac12kA^2=\frac12m\omega^2A^2$。

## 适用条件

- 弹簧振子忽略阻尼和驱动力时是简谐振动。
- 单摆只有小角度近似下才是简谐振动。
- 求初相必须同时满足初始位移和初始速度。

## 解题步骤

1. 找平衡位置作为位移零点。
2. 写回复力或加速度。
3. 求 $\omega,T,f$。
4. 用初始条件求振幅和初相。
5. 能量题用动能、势能互换。

## 易错点

- 振幅是最大位移，不是任意时刻位移。
- 速度最大在平衡位置，加速度最大在端点。
- 相位差决定振动状态，不能只看时间差。
- 单摆周期与振幅无关只在小角度近似成立。
