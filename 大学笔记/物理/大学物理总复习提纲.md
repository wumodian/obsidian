# 大学物理总复习提纲

> 返回 [[大学物理习题与解析]]。本页用于把章节、知识点、题型方法串成一条复习路线。

## 复习顺序

1. **质点力学**：先掌握 [[运动学]]，再进入 [[质点力学综合]]、[[动量定理与动量守恒]]、[[功与机械能]]。
2. **刚体力学**：先会算 [[转动惯量]]，再用 [[力矩与转动定律]] 建方程，最后处理 [[角动量定理与守恒]]。
3. **振动与波动**：用 [[简谐振动]] 理解相位、周期和能量，再用 [[机械波与驻波]] 处理传播和叠加。
4. **波动光学**：重点区分 [[光的干涉]] 与 [[光的偏振]] 的条件、公式和典型判断。
5. **静电场**：先复习 [[静电场]] 的叠加思想，再掌握 [[高斯定理]]、[[电场强度与电势]]、[[导体的静电平衡]]。
6. **稳恒磁场**：按 [[稳恒磁场]]、[[安培环路定理]]、[[载流导线的磁场与安培力]]、[[带电粒子在磁场中的运动]] 的顺序复习。
7. **电磁感应**：先判断磁通量是否变化，再用 [[法拉第电磁感应定律]] 和 [[电磁感应]] 处理大小与方向。
8. **量子物理**：先抓 [[黑体辐射与能量子]]，再用 [[光电效应]] 理解光子能量和阈值条件。

## 刷题策略

- **第一遍**：看题后先判断题型，不急着算，写出对应知识点。
- **第二遍**：遮住答案，只写关键方程或判断理由。
- **第三遍**：把错题按“概念错、公式错、方向错、单位错、题图错”归类。
- **考前复盘**：优先看 [[大学物理易错点与方法]] 和 [[大学物理公式总表]]。

## 自查清单

- 是否能说明每个公式中符号的含义？
- 是否能判断什么时候用守恒，什么时候不能用守恒？
- 是否能区分矢量、标量、大小和方向？
- 是否能把题目条件转成图示、方程或边界条件？
